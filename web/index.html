모델 원예측 확률

아래 막대는 안전 보정 전 모델이 계산한 중요도별 확률입니다.

    판단 기준

    입력 문장을 분석하면 기준 설명이 표시됩니다.

    중요도별 예시 민원

    각 등급의 판단 차이를 확인할 수 있는 테스트 문장입니다.

    중요도 1

    단순 문의·정보 확인

    중요도 2

    개인 불편 중심 일반 요청

    중요도 3

    구체 조치가 필요한 일반 민원

    중요도 4

    공공안전·다수 피해 우선 처리

    중요도 5

    생명·재난·안보 즉시 대응

    연구 목적

    단순 문의와 긴급 민원을 구분하여 처리 지연을 줄이고, 생명·안전·다수 피해와 관련된 민원을 우선 확인하도록 돕는 모델입니다.

    모델 방식

    민원 문장을 문자 n-gram으로 임베딩하고 선형 분류 모델로 중요도 1~5단계를 예측합니다. 웹 데모는 브라우저에서 직접 실행됩니다.

    주의 사항

    본 결과는 행정 처리 보조용 실험 모델이며, 실제 우선 처리 여부는 담당자의 최종 검토와 기관 기준을 함께 적용해야 합니다.
